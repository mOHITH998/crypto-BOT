# Crypto-Bot 
- crypto Telegram bot using Node.js

![commands example](Img/crypto-img.jpg)

### What is it??
- This Bot is Created to Track the Prices of Crypto currencies

### How to Run 

- To clone Repository:
```bash
$ git clone [Repo URL]
```

- To install Dependencies:
```bash
$ npm install
```
- To Run:
```bash
$ npm start
```
